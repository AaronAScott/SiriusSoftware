# Sirius eFile

Sirius eFile is an electronic document-filing program.  Its purpose is to both localize your important documents, and free you from holding paper copies in huge filing cabinets.

**Note: this program requires LocalDB version 2019 or higher.**  If this is not installed on your computer, you can download it via this link: https://go.microsoft.com/fwlink/?linkid=2215160.  Use only SQLLocalDB.msi, which will extract from the download.

## Features

* Documents can be stored in both permanent folders, as well as folders by year.
* Deleted documents can be recovered for up to 6 months.
* Documents can be pasted from the clipboard as images, or take from a scanner.
* Redactions and "Sticky Notes" can be added as layers to a document, without changing the underlying document.
* Document "Cabinets", both permanent and by year, can be locked by the supervisor.
* Supervisor-determined access to documents.
* An optional "Copy"  watermark can be added to any exported, emailed or printed copies of documents.
* Built-in "Shredder" function allows documents to be destroyed beyond recovery by disk utility programs.
* Automatic document de-skewing.
* Supports both WIA and Twain scanners.
