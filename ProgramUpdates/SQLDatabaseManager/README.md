# SQL Database Manager

A lightweight, Access‑style tool for designing, creating, and manipulating **SQL Server** databases — without the overhead, clutter, or ceremony of SSMS. SQL Database Manager provides the immediacy of table‑centric editing with the reliability and future‑proofing of SQL Server LocalDB.

## Overview

SQL Database Manager is built for developers and power users who want a fast, intuitive way to work with SQL Server databases. It replaces the aging Access workflow with a modern, self‑contained environment that supports:

- Creating and modifying databases  
- Designing tables, indexes, and relationships  
- Editing data directly in grid views  
- Executing ad‑hoc SQL  
- Managing SQL Server LocalDB instances  
- Importing and exporting data  

All without requiring SQL Server Management Studio or full SQL Server installations.

## Why This Exists

Access was great — until it wasn’t.  
LocalDB is great — but not user‑friendly.

SQL Database Manager bridges the gap:

- **Access‑like immediacy**  
- **SQL Server reliability**  
- **No deprecated engines**  
- **No fragile shared‑file model**  
- **No installers or admin rights required**

It’s the tool Access should have grown into.

## Key Features

- **Database Designer**  
  Create and modify tables, keys, constraints, and indexes through a clean UI.

- **Data Editor**  
  Open a table and edit rows directly — no boilerplate SQL required.
