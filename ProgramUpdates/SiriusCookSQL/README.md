# Sirius Cook SQL

Sirius Cook is a recipe management program, with roots going back over 30 years to v1.000.  It stores recipes locally, in an SQL Server LocalDB database, and provides features to scale a recipe up or down, email directly, or format for texting, and exchanging recipes via an XML format.

**Note: this program requires LocalDB version 2019 or higher.**  If this is not installed on your computer, you can download it via this link: https://go.microsoft.com/fwlink/?linkid=2215160.  Use only SQLLocalDB.msi, which will extract from the download.

## Features

- Recipes can be organized in "cookbooks" arranged on a "bookshelf".  Alternatively, you can store them all in the default cookbook.
- Recipes can be automatically scaled up or down in size, based on the number of servings, or by a percentage.
- Images of a finished dish can be attached to a recipe.
- Recipes can be emailed in various formats: RTF, plain text or HTML.
- Recipes can be printed out for use or to give away, or viewed in large format directly on the screen.
